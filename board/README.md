# MagicMirror树莓派扩展板

提供本项目的树莓派扩展板的.sch以及.pcb文件。

### 功能

此扩展板为树莓派提供温度传感器、光强传感器。

### 元件清单

* ADS1115
* DS18B20
* 光敏二极管（型号任意）
* 0603封装电阻（1k, 4.7k, 47k）
* 0805封装电容
