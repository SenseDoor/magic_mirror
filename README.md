# magic_mirror
使用树莓派完成的魔镜项目，功能包括人脸识别、Web交互、温度传感、光强传感。
## 使用指南
* 使用前准备
  将代码clone到树莓派中；制作pcb板并焊接，连接到树莓派上。
* 给树莓派加电，运行main.py（可设置为开机自启动，参考https://blog.csdn.net/qq_33287871/article/details/99687067）
